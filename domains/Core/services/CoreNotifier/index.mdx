## Overview

The Core Notifier is a reactive service that subscribes to budget lifecycle events from the [[container|CoreEventStore]]. When a budget threshold is reached or exceeded, it dispatches user-facing notifications (e.g. push notifications or in-app alerts).

It does not emit any domain events back into the event store — notifications are fire-and-forget side effects.

```
BudgetReached   → [CoreNotifier] → BudgetThresholdReached notification (out-of-band)
BudgetExceeded  → [CoreNotifier] → BudgetExceeded notification (out-of-band)
```
