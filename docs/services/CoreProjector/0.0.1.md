## Overview

The Core Projector is the read-side component of the Core domain. It consumes all events from the [[container|CoreEventStore]] — written by both the [[service|CoreAggregate]] and the [[service|CoreEventInjector]] — and maintains three read models:

- **[[container|TransactionView]]** — per-transaction projection optimised for list and search queries.
- **[[container|BudgetView]]** — per-budget projection tracking spending progress and threshold status.
- **[[container|DashboardView]]** — aggregated financial summary for dashboard-style rendering.
