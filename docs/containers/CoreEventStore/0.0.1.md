## Overview

Core Event Store is the append-only event table for the Core domain. The events are emitted either [[service|CoreAggregate]] or [[service|CoreEventInjector]].

The [[service|CoreProjector]] and [[service|CoreNotifier]] consume events from this store to build read models and dispatch notifications.
